# Apisix_Crack
## 概述
Apisix系列漏洞探测利用工具，包括未授权访问漏洞（CVE-2021-45232）和默认秘钥漏洞（CVE-2020-13945），支持批量探测利用。
## 使用方法
批量探测：文件中逐行写入需要检测的URL，参数-TF 指定文件 -t 并发数量

单个目标探测：-T http://x.x.x.x
